Calculate the most optimal enchantment order in Minecraft (in anvil). Select the item, select the enchantment, change the enchantment level. Once everything is set, click search. To remove an item, select the empty option in drop down list. It is optional to select the output target enchantments.
Reference Page: Minecraft Fandom Wiki
Github: Github
Capable to search in :
enchanted book + enchanted book
item + enchanted book
enchanted item + enchanted item
enchanted item + enchanted book
books with more than 1 enchantment
combining 3 to 8 item/books
Out of Scope:
renaming/repair
combining more than 8 item/books
enchanted books with non zero prior penalty
Output Target:
Search
