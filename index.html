Minecraft Enchantment Order Calculator
Reference Page: Minecraft Fandom Wiki
Github: Github
Calculate the most optimal enchantment order in Minecraft. There are 2 Modes available:
(1) Combining multiple books (with 1 enchantment) to an item: click the item in "You Want" and select the enchantments.
(2) Combine multiple enchanted items and books together: add items one by one in "You Have" and select the enchant each.
In (2), you can specify the final outcome with enchantments, select them in "You Want".

To remove an item/enchant, select the blank option, or set the level to 0.
Once everything is set, click search. (select at least 3 items/enchantments to search)
You can save the search result as image, and reload it by dragging the image into this page.
