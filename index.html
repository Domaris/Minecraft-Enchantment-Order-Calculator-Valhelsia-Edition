Calculate the most optimal enchantment order in Minecraft (in anvil). There are 2 use cases fulfilled by this tool:
(1) If you want to combine books with 1 enchant to an item, click the item in "You Want" and select the enchantments there.
(2) If you want to combine multiple items/books, add them in "You Have" and select the enchantment on each item/book there.
In (2), if you want your final outcome to have specific enchant level, select them in "You Want".

To select an enchantment, click on and change the number next to enchant name. You can also click on "max" to set the level.
In (2), you have to first add the enchant on your item one by one in "enchant with" for all item you have.
To remove an item/enchant, select the blank/delete the enchantment, or set the level to 0. Once everything is set, click search.
